## ⚠️ 先行gym制作からの学び（必読・優先度最高）

1本目のgym.html（FORGE）制作で発覚した致命傷を、このCompassでは**最初から回避**すること：

### 写真の扱い（肖像権リスク）
- **講師・受講生インタビューの人物写真は、Unsplash等のフリー素材を絶対に使わない**
- 架空人物のプロフィール枠に他人の実写を当てるのは、**Unsplash規約違反＋肖像権侵害**
- Compassの場合は **講師ポートレート・受講生インタビュー3〜5名・FV俯瞰人物** が該当
- **正しい実装**：`#f5f1ea` or `#faf8f3` 単色背景＋斜めストライプテクスチャ＋「— 講師 / 撮影予定」スパン
- 例（gymで採用された CSS パターン）：
  ```css
  .person-photo{background:#f5f1ea;position:relative}
  .person-photo::before{content:"";position:absolute;inset:0;
    background:repeating-linear-gradient(45deg,#ede5d3 0 8px,#f5f1ea 8px 16px);opacity:.6}
  .person-photo span{position:absolute;bottom:18px;left:18px;
    font-family:var(--ff-num);font-style:italic;font-size:11px;
    letter-spacing:.22em;color:var(--c-mute);text-transform:uppercase}
  ```
- 本・万年筆・手帳・コーヒー・観葉などの**人物が写っていない俯瞰素材**は Unsplash 可（HTML末尾に差し替えTODOコメント残す）

## 4. AI感を消す原則（最優先）

### 禁止語（必ず置換）

| AI語 | 置換 |
|---|---|
| 本当の自分／なりたい自分／自己実現／可能性を引き出す | （削除して具体ケースへ） |
| マインドセット／人生を変える／変革／パラダイムシフト | （削除） |
| 最適な／最高の／究極の／確かな／質の高い | のべ◯人／3年やってる |
| 圧倒的な実績／豊富な経験 | のべ◯人／7年やってる |
| あなたに最適な／お一人おひとりに | あなた／田中さん（仮名） |
| 〜を実現します／〜を提供します／〜をサポートします | 〜できます／週1で30分話します |
| ライフスタイル／ニーズ／ソリューション／メソッド | 暮らし／困りごと／やり方／◯◯式 |
| 至福の／上質な／ワンランク上の／豊かな人生 | （削除） |
| 寄り添う／伴走する／一緒に歩む | 横で見てます／週1で話します |

### 禁止文型

1. **冒頭の問いかけ三連**「〜なお悩みはありませんか？」型 → 「『このままでいいのかな』その違和感は、次のステージへのサインです」型へ
2. **3点並列の体言止め**「あなたの強み・あなたの未来・あなたの可能性」 → 削るか、最低1つは口語で崩す
3. **抽象→具体テンプレ**「一人ひとりに最適なコーチングをご提案。例えば〜」 → いきなり具体ケースから
4. **結びの誘導定型**「あなたも理想の人生を手に入れませんか？」 → 削除して「無料カウンセリング、月10名限定で受けてます。」型へ
5. **「お客様の声」AI生成丸出し**：「素晴らしいコーチングでした！(30代女性)」 → 「正直、コーチングって怪しいと思ってました。3回目のセッションで、初めて自分の口で『私はこれが好き』って言えた日があって。(田中さん・34・経理)」型

### 人間味を出す7つの手筋（1ページ内に最低5つ仕込む）

1. **一人称**：「私」「うちのプログラム」「鈴木です」を文中に最低1回
2. **固有名詞・固有数字**：「のべ312名」「3ヶ月で12回」「Slackで返信は24時間以内」
3. **ためらい・限定**：「全員には合いません」「即決される方は、お断りしています」
4. **断言**：「週1で30分話します」「宿題は出しません」と言い切る
5. **口語の挟み込み**：「で、」「ぶっちゃけ」「ですよね」を要所に1つ
6. **失敗談・反対意見**：「独立した最初の年は、月収3万円でした」
7. **読点で粘る**：句点で短く切らず、読点で息を作る文を1セクションに1つ

### ビジュアルAI臭の禁止

- 中央寄せ3カラム×等間隔の「特徴3つ」テンプレ → ジグザグ配置や非対称分割（5:7）へ
- shadow-md（0 4px 6px）の量産 → 影は1箇所だけ、他は罫線で
- border-radius 16px 全要素統一 → カード6/ボタン4/写真0
- fade-in-up全乗せ → ヒーローのみ
- hover時 scale(1.05) → 色変化200ms or 下線伸長のみ
- **SaaS的グラデ・ダッシュボードUI画像 禁止**（コーチ業に冷たい）
- unDraw/Storyset風イラスト → 手書きノート風の図解・SVG手描き
- **AI生成人物（指の融合、白人女性の白背景）禁止**。受講生インタビューは実写優先
- 紫×シアン/ティール×オレンジ/ネオン → 禁止
- backdrop-blur-xlのガラス風 → 禁止
- 「導入1,000社」風の根拠なきカウントアップ → 禁止

## 5. 和モダン指針（紙バリアント）

### レイアウト

- グリッド12カラム、コンテンツ最大幅 1040〜1120px（長文ゆえ少し狭め＝読みやすく）
- 本文ブロックは最大幅 680px（読了率優先）
- セクション縦余白：PC 120〜180px / SP 80〜120px、**意図的に不揃いに**
- 非対称分割（5:7）を1セクションは入れる
- 章番号を縦組みで余白に置く（左側、Shippori Mincho 800、金茶）
- 引用ブロック多用：左に1px藍罫＋字下げ、Shippori Mincho 500

### 装飾

- 罫線は0.5pxか1pxどちらかに統一、色は `--border` 鳥の子
- 紋様（雲・霞）はフッター帯1箇所、不透明度8〜10%
- 紋様タイル背景・印鑑風・桜扇梅マーク禁止
- **紙の繊維テクスチャを 3% 不透明度で全面**（紙の和モダンのコア）
- バッジは-2degの回転を1〜2箇所だけ
- 章番号を縦組みで添える＝「書物感」の演出

### タイポ細部

- 本文 line-height: 1.9
- 句読点ベタ詰めで重い箇所は `font-feature-settings: "palt"`
- 引用ブロックは字下げ＋左罫
- 章番号は縦組み（writing-mode: vertical-rl）

### アニメーション

- fade-in + translateY(20px) を 700ms ease-out のみ
- ヒーローのみ控えめパララックス（10%以内）

### 写真ディレクション

- **受講生インタビュー写真は実写優先**（AI生成不可）
- 講師ポートレートは斜光横顔・低彩度
- 抽象背景はAI可、`35mm film / grain / shadow / low saturation` 指定
- ストックの「PCで微笑む白人女性」「握手」「指差すスーツ」禁止
- 小物（本・万年筆・手帳・コーヒー・観葉）の俯瞰写真を要所に
- カラーは彩度-20〜-30、シャドウ持ち上げない、自然光

## 6. セクション構成（長文LP・ストーリー型）

### 固定ヘッダー
- ロゴ（左）／簡易ナビ（Compass/メソッド/受講生/料金/講師/FAQ）／右にCTA `[無料カウンセリング]`
- スクロール時に背景を `#faf8f3` の95%不透明＋細罫線
- モバイル下部追従：LINE登録（藍）と無料カウンセリング（煉瓦）

### 01 FV（ヒーロー）
- 背景：白練。窓辺で手帳を開く女性の俯瞰写真（顔は半分外）を端まで bleed、4:5縦長
- H1：「なんとなく不安な毎日から。」（Shippori Mincho 700 / 60px）
- サブ：「自分の"方位"を、3ヶ月で見つける。」
- 実績バッジ3つ（横並び）：受講生のべ312名 / 継続率92% / メディア掲載5媒体
- CTA2つ：`[無料カウンセリングを予約]`（煉瓦）／ `[15の質問で診断する]`（藍枠線）
- 章番号「01」を余白に縦組みで（金茶）

### 02 共感（不安リスト）
- 「こんな毎日に、心当たりがあれば。」
- 不安リスト6項目（チェックリスト＋短い独白）：
  - 「朝、目覚ましを止めるのが、年々遅くなっている。」
  - 「同期がキラキラ見える日と、何も感じない日の差が、激しい。」
  - 「『このままでいいのかな』が、口グセになっている。」
  - 「やりたいことを聞かれて、答えに詰まる。」
  - 「転職サイトを開いては、閉じる。」
  - 「夜、眠る前にだけ、何かを変えたくなる。」

### 03 講師ストーリー（共感の橋）
- 「私も、同じ場所に立っていました。」
- 講師の原点ストーリー、500〜800字、口語、一人称
- 「3年前、毎朝のZoomを切ったあと、トイレで泣いていた時期がありました。」型の具体描写から
- 引用ブロック1〜2箇所
- 講師の手元写真（万年筆・手帳）

### 04 未来像（理想の3ヶ月後）
- 「3ヶ月後の、ある朝。」
- 朝・仕事・週末の3シーンを具体描写（各150〜200字）
- 「目覚ましが鳴る前に、目が覚めるようになりました。」など固有エピソード
- イラストではなく実写（朝食・PC・手帳）

### 05 メソッド（羅針盤メソッド®）

## 10. 法的禁則（コーチング業界・最重要）

### 特商法表記必須
- フッターに**特商法表記**へのリンク（運営者名・所在地・連絡先・返金条件・中途解約条件・支払方法）
- LP内に「特定商取引法に基づく表記」リンクを必ず

### 景表法 優良誤認・有利誤認

絶対NG：
```
絶対, 100%, 必ず, 保証（条件なしで）,
業界No.1, 日本一, 最高, 最大級, 史上初,
全員成功, 誰でも必ず, 最短最速,
年収◯倍（実例として書くなら「個人差注記」必須）
```

OK表現：
- 「受講生の85%が◯◯を実感（自社アンケート n=◯）」のように **母数明記**
- 「ICF認定コーチ」（資格は事実）
- 「のべ312名」（実数）
- 「3年連続、満足度4.5/5」（出典添える）

### 景表法 二重価格
- 通常価格表示は「過去価格8週間以上」が条件、注記推奨

### 薬機法（健康系コーチでなくても注意）
- 「治る」「改善する」（身体・心の症状）禁止
- 「整える」「サポート」へ

### 受講生の声
- **「※成果には個人差があります」** 必ず添える
- 「年収が3倍になりました」断定は注記必須

### 無資格名乗り
- 「カウンセラー」「セラピスト」名乗りは要注意（民間資格でも明示）
- 「コーチ」自体は資格不要だが、ICF等の資格があれば明示推奨

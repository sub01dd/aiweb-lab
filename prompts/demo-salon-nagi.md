## ⚠️ 先行gym制作からの学び（必読・優先度最高）

1本目のgym.html（FORGE）制作で発覚した致命傷を、この凪では**最初から回避**すること：

### 写真の扱い（肖像権リスク）
- **施術者・お客様の声・施術風景の人物写真は、Unsplash等のフリー素材を絶対に使わない**
- 架空人物のプロフィール枠に他人の実写を当てるのは、**Unsplash規約違反＋肖像権侵害**（被写体の権利は写真家ライセンスとは別物）
- 凪の場合は **施術者の顔・施術中の人物・お客様の声の顔写真** が該当
- **正しい実装**：`#1a1a1d` or `#e2ddd2` 単色背景＋斜めストライプテクスチャ＋「— 施術者 / 撮影予定」スパン
- 例（gymで採用された CSS）：
  ```css
  .trainer-photo{background:#f5f1ea;position:relative}
  .trainer-photo::before{content:"";position:absolute;inset:0;
    background:repeating-linear-gradient(45deg,#ede5d3 0 8px,#f5f1ea 8px 16px);opacity:.6}
  .trainer-photo span{position:absolute;bottom:18px;left:18px;
    font-family:var(--ff-num);font-style:italic;font-size:11px;
    letter-spacing:.22em;color:var(--c-mute);text-transform:uppercase}
  ```
- 院内・道具・木・畳・観葉などの**人物が写っていない素材**は Unsplash 可（HTML末尾に差し替えTODOコメント残す）

## 4. AI感を消す原則（最優先）

### 禁止語（必ず置換）

| AI語 | 置換 |
|---|---|
| 根本改善／根本治療／根治／完治 | （**法的にも禁止**。「楽になる」「軽くする」「整える」へ） |
| 治る／治療／効果保証 | （**法的にも禁止**。「お悩みにアプローチ」「緩和」） |
| 身体本来の／自然治癒力／お身体と向き合い | （削除して所作の描写へ） |
| 最適な／最高の／確かな技術／質の高い | （削除して具体ディテール） |
| 圧倒的な実績／豊富な経験／きめ細やかな施術 | のべ◯人診た／10年やってる／60分かけて |
| お客様一人ひとりに寄り添い | あなた／田中さん（仮名） |
| 〜を実現します／〜を提供します／〜をサポートします | 〜できます／〜します／一緒にやります |
| 優しい施術／心地よい空間／至福のひととき | ボキボキしません／木の匂いがします |
| 健康と美しさを叶える | （削除） |
| トータルケア／ライフスタイル／メソッド | （具体行動／暮らし） |

### 禁止文型

1. **冒頭の問いかけ三連**「〜なお悩みはありませんか？」 → 「鏡に映る肩を、もう撮りたくない。」型へ
2. **3点並列の体言止め**「丁寧なカウンセリング・優しい施術・心のこもったアフターケア」 → 削るか、最低1つは口語で崩す
3. **抽象→具体テンプレ**「お一人おひとりに最適な施術をご提案。例えば〜」 → いきなり具体ケースから
4. **結びの誘導定型**「あなたも理想のお身体を手に入れませんか？」 → 削除して「電話、9時から18時まで出ます。私が出ます。」型へ
5. **中点「・」濫用** 1ページに1回まで（「食事・運動・睡眠・メンタル」型を避ける）
6. **「お客様の声」AI生成丸出し**：「丁寧な対応で安心できました！（30代女性）」 → 「ボキボキされるのが怖くて、3軒目でやっとここに来ました。1回で『あ、これは違う』ってなった。(田中さん・42・経理)」型

### 人間味を出す7つの手筋（1ページ内に最低5つ仕込む）

1. **一人称**：「私」「うちの院」「鈴木です」を文中に最低1回
2. **固有名詞・固有数字**：「のべ4,200人」「西淀川区・千舟3丁目」「腸腰筋」「肩甲下筋」
3. **ためらい・限定**：「全員に効くわけではないです」「ボキボキされたい方には向きません」
4. **断言**：「ボキボキしません」「指圧だけです」と言い切る
5. **口語の挟み込み**：「で、」「ですよね」「ぶっちゃけ」を要所に1つ
6. **失敗談・反対意見**：「最初の3年は、口コミでなかなか広がらなくて」
7. **読点で粘る**：句点で短く切らず、読点で息を作る文を1セクションに1つ

### ビジュアルAI臭の禁止

- 中央寄せ3カラム×等間隔の「特徴3つ」テンプレ → ジグザグ配置や非対称分割（5:7）へ
- shadow-md（0 4px 6px）の量産 → 影は1箇所だけ、他は罫線で
- border-radius 16px 全要素統一 → カード6/ボタン4/写真0
- fade-in-up全乗せ → ヒーローのみ
- hover時 scale(1.05) → 色変化200ms or 下線伸長のみ
- unDraw/Storyset風イラスト → 写真＋手描き丸囲み（SVGで線をブレさせる）
- **AI生成人物（指の融合、白衣の手）禁止**。施術者・お客様の声は実写優先、ない場合はプレースホルダー
- 紫×シアン/ティール×オレンジ/ネオン → 禁止
- backdrop-blur-xlのガラス風 → 禁止
- 紫グラデ・スピリチュアル系装飾 → 怪しさを増幅するため禁止

## 5. 和モダン指針（静バリアント）

### レイアウト

- グリッド12カラム、コンテンツ最大幅 1080〜1160px（鋼より少し狭め）
- セクション縦余白：PC 140〜200px / SP 96〜120px、**意図的に不揃いに**
- 非対称分割（5:7や3:8）を1セクションは入れる
- 余白の中に小さな手書きメモを1箇所（「※施術時間は約60分」を斜め-2degで）
- 画像は縦長 4:5 を1〜2枚混ぜる（和の比率）

### 装飾

- 罫線は0.5px統一、色は `--border` で1段階、必要なら藍透過 `rgba(44,74,110,.12)`
- 紋様（青海波 or 雲）はフッター帯1箇所、不透明度8〜12%
- 紋様タイル背景・赤い印鑑風スタンプ・桜扇梅マーク・暖簾写真は禁止
- SVGノイズ3〜5%を背景に重ねて紙質感
- 雲/霞のSVGフェード境界をセクション切れ目に1〜2箇所

### タイポ細部

- 本文 line-height: 1.95〜2.0（「呼吸」を作る）
- 句読点ベタ詰めで重い箇所は `font-feature-settings: "palt"`
- 縦組みアクセント：章番号・キャプションだけ縦に置く

### アニメーション

- fade-in + translateY(20px) を 700〜900ms ease-out のみ
- ヒーローのみ控えめパララックス（10%以内）
- 動かさない勇気が「静」のコア

### 写真ディレクション

- **施術者・お客様の声は実写優先**（AI生成不可）
- 施術風景は手元・道具・布のクローズアップを多用、顔は控えめ
- 院内写真（受付/個室/施術ベッド/外観）4枚以上
- カラーは彩度-30〜-40、シャドウ持ち上げない、自然光
- 抽象的なヒーロー背景はAI可（Midjourney等）、`35mm film / grain / shadow / low saturation` 指定、`clean / modern / minimalist` 禁句
- ストックの「優しく微笑む白人女性」「白衣で手を組む」禁止
- ビフォーアフター写真は **掲載しない**（医療広告ガイドライン準拠）

## 6. セクション構成

### 固定ヘッダー
- ロゴ（左）／ナビ（凪/お悩み/施術/料金/施術者/アクセス）／右に2CTA `[LINEで予約]` `[☎ tap-to-call]`
- スクロール時に背景を `#f7f3ec` の95%不透明＋細罫線
- モバイル下部追従：LINE予約（藍）と電話（白＋藍枠線）

### 01 FV（ヒーロー）
- 背景：生成り。施術者の手のクローズアップ（顔なし）を端まで bleed、縦4:5
- H1：「もう、付き合うのをやめる。」（Shippori Mincho 700 / 56px）
- サブ：「慢性的な肩こり・腰痛と、寄り添う60分。慢性症状専門サロン 凪。」
- 安心バッジ3つ（縦並びor非対称）：完全予約制／完全個室／女性施術者
- 初回価格：`初回 ¥4,980`（通常 ¥8,800 は二重線）
- CTA2つ：`[LINEで予約]`（藍）／ `[☎ 03-XXXX-XXXX]`
- 「西淀川区・千舟3丁目／阪神千船駅 徒歩3分」を小さく添える
- パララックスは10%以内

### 02 悩みの言語化
- 「こんな日が、続いていませんか。」
- 共感5行（チェックリストではなく独白寄り）：
  - 「朝、首が回らない。」
  - 「夕方には、肩が石みたいに重い。」
  - 「市販の湿布じゃ、もう追いつかない。」
  - 「他の整体に行っても、3日でまた戻る。」
  - 「ボキボキされるのが、怖い。」
- 「※こんなお悩みはありませんか？」型は禁止

### 03 なぜ繰り返すのか
- 「あなたの肩こりが、また戻る理由。」
- 3つの視点（姿勢／筋膜／生活習慣）。横並び3カラムは禁止 → ジグザグor非対称
- 各120〜180字、医学用語は最小限、口語で
- 「治る」「効果」表現は使わない

### 04 凪の5つの特徴
- 「凪が選ばれる、5つの理由。」
- 5項目：
  1. 慢性症状だけに特化して10年（具体年数）
  2. 国家資格保有（柔道整復師 or 鍼灸師）
  3. 完全予約制・完全個室
  4. 女性施術者ワンオペ・担当固定
  5. 初回は60分カウンセリングと施術
- 縦リスト or 非対称グリッド、各説明200字

### 05 施術の特徴（体感の言語化）
- 「眠ってしまうほどソフトな圧。」
- ボキボキしない／指圧主体／触れる場所（腸腰筋・肩甲下筋など具体筋名）を1〜2個固有名詞で

### 06 お客様の声
- 6〜10件、年代・症状・地域別タグ
- 各300〜500字、必ず疑念から始める：「ボキボキされるのが怖くて、3軒目でやっとここに来ました」

- ※「治った」「完治」表現は声でも使わない

### 07 改善までの目安（通院モデル）
